内容要約

はじめに

  • 「人々のつながりは意味を帯びており、意味は人・物・考えのネットワークにすぎない」という考え方を用いて、以下の3点の古典的な問題に答える。
    1. いかにして趣味が人々を分化していくか。
    2. いかにして意味の構造と社会関係の構造が共進化するか。
    3. いかにして意味の構造の変容が支配構造の変容を形作るか。
  • アートワールドは、意味付与や事物の質に関する知覚によって関係するコミュニティから構成される。

アート・ワールド

  • Dantoによるアートワールドのフォーマルな記述
    • アートワールドに存在する作品（事物）の集合と、それに付与される美点の可能性の集合を考える。
    • それぞれのアート作品にそれぞれの美点があるかどうかをなし＝0、あり＝1の2値で評定していく。
    • 例えば、以下のような美点×作品の行列を M とすると
美点1 美点2 美点3 美点4
作品A 1 0 1 1
作品B 0 0 1 1
作品C 1 0 0 1

MMt\bf{M} \bf{M}^tMMtが作品×作品の行列となる。ここで各要素は各作品を共通して持っている美点の数を表す。

作品A 作品B 作品C
作品A 3 2 2
作品B 2 2 1
作品C 2 1 2

MtM\bf{M}^t \bf{M}MtMが美点×美点の行列となる。ここで各要素は各美点を共通して持っている作品の数を表す。

美点1 美点2 美点3 美点4
美点1 2 0 1 2
美点2 0 0 0 0
美点3 0 0 2 2
美点4 1 0 2 3
  • アートワールドが豊かになっていく過程でいかにして行と列が追加・消去されていくかは、社会学的な関心である。

「アート・ワールド」での位置

  • Dantoはアートワールドが社会的過程からは遊離しているものと捉えた。
    • 経験的なアートの社会学の多勢も、Danto的なアートの世界が「他の社会」とは独立して成立していると考えていた。
    • Bourdieuは、アート生産の場は、その中での位置取りをめぐる闘争であるとした。
      • Danto的に言えば、アートの世界は、他のものが占める位置の価値を下げる形で、行列の中に体系的に位置を占めていく競争の場である。
      • 作品が占める美学的な位置は、作品だけでなく、作家個人にも付与されるものである
    • Beckerのアートワールドは人々のネットワークである。
      • 美学的特性は社会構造に付随する。
      • Beckerは美学的に関連のある特性の宇宙(e.g. ジャンルやカノンなど)を定義するものとしての社会的交換のパターンを重視。
        • どのような事物を正統と見なすかによって、人々は下位集団へと分けられる。
  • アートワールドの全ての可能性が等しく実際に実現されない限り、アートワールドは意味のある構造である。
  • 結局のところアートワールドは、人によってつながっている事物のネットワーク(=Bourdieu)としても、事物によってつながっている人のネットワーク(=Becker)としても見ることができる。

趣味と分化

  • Bourdieuの Distinction での主張：趣味は社会構造上の位置によって形作られる。
    • ↔個人の好き嫌いがどういうわけか自然に決まるという個人的趣味という直感的なアイデア)
    • Bourdieuにとって、趣味をめぐる闘争とは客観的な社会構造上の地位の意味をめぐる闘争である
  • 「共有された理解」をもつコミュニティを特定する方法
    • GoldbergのRCA(Relational Class Analysis)
  1. RCAは、データセット内の観測の各ペアについて、関係性の尺度を計算する。その結果、完全グラフ、無向グラフ、および加重グラフが作成される。
  2. 重みが統計的に有意でないグラフエッジは削除され、残っているエッジは絶対値で変換される。
  3. RCAは、グラフ分割アルゴリズムを使用して、グラフを同様の観測結果のサブグループに分割する。
    （Goldberg 2011：1405）

意味の構造と関係の共進化

  • Fraser(1985)の女性の衣服の変化に関する分析
    • 20世紀半ばの米国では、衣服の着用は、社会的アイデンティティ（すなわち、主に年齢、婚姻状況）のシグナルであった。
    • 女性の社会的地位による衣服の着用への制限がなくなったことで、服装による「自己表現」が可能になった。
    • 衣服は、「どのように見られるか」から「どのような意味か」によって選択されるようになった。
  • Quentin Van Doosselaere(2009)による中世ジェノヴァを起点とする商業資本主義の台頭の説明
    • commendaと呼ばれる初期のエクイティ協定の浸透 -> 貿易の拡大に成功。
    • 分析の基本的方針：ネットワーク構造と、社会的アイデンティティの変化を時系列的に観察。
      • commendaに参加する個人のネットワークを分析
        • ネットワークには貴族による密なコアが形成されていたが、ランダムネットワークとの比較を行うと、クラスタリングの程度が低かった(http://eclectic.ss.uci.edu/~drwhite/Conf/Doos_conf_complexity.pdfにプレプリントあり)。
          • これは非市場的な過程によって紐帯が形成されたことを示している。
        • 中世では貴族の中での利益を高めるために形成された紐帯が、後に商人たちの成功にとって欠かせない要素となった。

都市のパトロンは、伝統的な資源を少数の一族に合わせた商業活動に変え、中世の封建権からルネッサンスの商業支配階級への突然変異を完成させていた。逆に、中央集権的および階層的集団の分裂によって特徴付けられる封建的過渡的社会秩序は、人口の残りの部分からますます分離された単一の均質かつ密接なエリートネットワークの中で、激しい社会活動に再構成された (Van Doosselaere 2009:4)

意味の構造の変容と支配の構造の変容

  • Danto的な観点からすれば、列の破壊とはすなわち支配である。
    • Beckerによれば、特定の意味構造の美的可能性の哲学的正当化は、集団行動の特定のパターンを調整し、それらを正統化し、社会構造を再構成するための補助的な役割を果たす。
    • 意味の構造変容は社会的支配の変容と関連している。
  • Bearman(1993) "Relations into Rhetorics"
    • 英国のジェントリー間の社会的・政治的行動の基盤としての親族の崩壊が、エリートなアイデンティティのための新たな時代をもたらし、政治的動員をどのようにもたらしたか。
    • 政治行動の指針としての親族の崩壊によって、宗教的同盟のイデオロギーによる郡を超えた動員が可能になった。

コメント

  • 著者について

    • Alix Ruleはアメリカの一般教書演説にLDAを用いた歴史研究で有名。ColumbiaのPeter Bearman門下。
    • Peter Bearmanは本文中のRelations into Rhetoricsや恋愛・性交渉ネットワークの研究(Bearman et al. 2004)、ネットワーク分析的なライフストーリーの研究(Bearman and Stovel 2000)で有名。

  • 本文を読んでの疑問

    • やたらと"in practice"や"practical"といった語を用いるのが気になった。
    • tastesによる人々のdifferentiationを明らかにしてそもそも何が嬉しいのか？
      • 趣味はなぜ社会学にとって重要か？
      • 社会経済的地位やその他の人口学的な変数を統制した上でも残る「趣味」の社会学的に重要な効果とは？
    • オブジェクトの知覚の傾向性(=審美眼？)が社会構造によって規定されているというのは理論的には確かに想定可能だが、あくまで諸要因のうちの一つとして考えるのが穏当。

  • 関係社会学(relational sociology, Emirbayer 1997)について

  • 趣味と差別的交際(differential association)に関する研究

    • 趣味(hobby)に基づくコミュニティは、成員の年齢に関する異質性が高い(McPherson 1983)
    • 趣味を通じて知り合ったカップルは、職業階層に関する異質性が高い(前嶋・打越 2017)。

  • 界を可視化・統計的分析するフォーマルな手法はかなり整備されている。

    • Bourdieu自身は対応分析を好んだが、MDSやネットワーク分析的手法も同様に有効。
      • パリ・コミューンの動員に近隣関係が果たした役割を明らかにしたGould(1991)や、メディチ家の支配体制に構造的空隙が果たした役割を明らかにしたPadgett and Ansell(1993)のように、ネットワーク分析は歴史研究にも使える。
